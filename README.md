# ULTRA GERBER VIEWER

High-performance Gerber / Drill Viewer with integrated realistic 3D PCB visualization.

Developed in Pure Python 3 + PyQt6 without external rendering engines.

## Features

- Ultra-fast Gerber rendering
- Realistic 3D PCB visualization
- Multi-layer PCB inspection
- Excellon Drill support
- Advanced raster/vector rendering
- Professional dark UI
- Measurement tools
- Aperture macro support

## Installation

pip install PyQt6

## Run

python UltraView1.py
